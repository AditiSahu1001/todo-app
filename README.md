This is a simple To-Do List desktop application built using "Python and Tkinter".  
The app helps users organize their daily tasks by allowing them to add, complete, and delete tasks through an interactive graphical interface.
- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Clean and simple user interface  
  "Technologies Used"
- Python  
- Tkinter (Python GUI library)

---

## How to Run the App
1. Make sure Python is installed on your system  
2. Download or clone this repository  
3. Run the following command:
   python todo_app.py
